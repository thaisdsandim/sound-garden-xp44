> Project developed for the Full-Stack Developer course at Gama Academy. Sound Garden is a fictional events website.

## :hammer: Functionalities

- `1`: Creat event
- `2`: Edit event
- `3`: Delet event
- `4`: List events
- `5`: Create bookings at events
- `6`: List bookings
